# VPN Infrastructure

This directory contains VPN setup and management tools.

## Available VPN Solutions

### WireGuard
Modern, fast, and secure VPN solution.

**Quick Start:**
```bash
cd wireguard
# Follow instructions in wireguard/README.md
```

**Features:**
- Easy setup and configuration
- Automated installation script
- VPN connection control (start/stop/restart)
- Health monitoring and maintenance
- Network configuration for wired connections
- Comprehensive documentation

**Documentation:** [wireguard/README.md](wireguard/README.md)

## Usage Overview

1. **Setup**: Install WireGuard and configure your VPN
2. **Control**: Manage VPN connections (start, stop, status)
3. **Maintain**: Monitor health and automatically restart if needed
4. **Configure**: Set up network routing and DNS for wired connections
