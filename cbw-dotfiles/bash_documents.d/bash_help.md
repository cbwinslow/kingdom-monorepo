# bash_help - Display help for bash functions

## Description
Main help system for Kingdom Monorepo bash functions. Displays a list of all available functions or detailed help for a specific function.

## Usage
```
bash_help
bash_help <function_name>
```

## Parameters
- `function_name` (optional): Name of the function to get help for

## Examples
```
# Show all available functions
bash_help

# Get help for a specific function
bash_help extract

# Get help for git functions
bash_help gcm
```

## Aliases
- `bh` - Short alias for bash_help
