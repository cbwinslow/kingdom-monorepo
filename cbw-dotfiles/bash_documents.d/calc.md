# calc - Quick calculator

## Description
Performs mathematical calculations using Python. Supports all Python math operations and functions.

## Usage
```
calc <expression>
```

## Parameters
- `expression` (required): Mathematical expression to evaluate

## Examples
```
# Simple arithmetic
calc "2 + 2"

# Complex calculation
calc "2 ** 10"  # 2 to the power of 10

# With parentheses
calc "(5 + 3) * 2"

# Division
calc "100 / 3"

# Modulo
calc "17 % 5"
```

## Notes
- Expression must be valid Python syntax
- Supports +, -, *, /, //, %, **, etc.
- Can use parentheses for grouping
