# note - Quick note taking

## Description
Quickly add timestamped notes to a daily notes file or open the notes file in your editor.

## Usage
```
note                    # Open notes in editor
note <note_text>        # Add note with timestamp
```

## Parameters
- `note_text` (optional): The note content to add

## Examples
```
# Open today's notes in editor
note

# Add a quick note
note "Remember to update the documentation"

# Add a longer note
note "Meeting notes: Discussed new feature implementation"
```

## See Also

- searchnotes - Search through notes
