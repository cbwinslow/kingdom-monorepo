# bash_search - Search for functions by keyword

## Description
Search through all bash functions and their documentation for a specific keyword or pattern.

## Usage
```
bash_search <keyword>
```

## Parameters
- `keyword` (required): Search term to look for in function names and documentation

## Examples
```
# Search for network-related functions
bash_search network

# Search for git functions
bash_search git

# Search for docker functions
bash_search docker
```

## Aliases
- `bs` - Short alias for bash_search
