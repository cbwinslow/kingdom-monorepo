# sysinfo - System information summary

## Description
Displays a comprehensive summary of system information including hostname, OS, kernel, uptime, load average, CPU, memory, and disk usage.

## Usage
```
sysinfo
```

## Examples
```
# Show system information
sysinfo
```

- topmem - Show top memory processes
- diskcheck - Check disk space
