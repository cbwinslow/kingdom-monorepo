# topcpu - Show top CPU processes

## Description
Displays the processes consuming the most CPU resources, sorted by CPU usage.

## Usage
```
topcpu [count]
```

## Parameters
- `count` (optional): Number of processes to show (default: 10)

## Examples
```
# Show top 10 CPU processes
topcpu

# Show top 5 CPU processes
topcpu 5

# Show top 20 CPU processes
topcpu 20
```
