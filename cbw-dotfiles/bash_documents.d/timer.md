# timer - Countdown timer

## Description
Starts a countdown timer for the specified number of seconds. Displays remaining time and alerts when finished.

## Usage
```
timer <seconds>
```

## Parameters
- `seconds` (required): Number of seconds to count down

## Examples
```
# 5 minute timer
timer 300

# 30 second timer
timer 30

# 1 hour timer
timer 3600
```
