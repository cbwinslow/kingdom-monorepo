# gcm - Quick git commit with message

## Description
Adds all changes and commits them with the specified message in one command.

## Usage
```
gcm <commit_message>
```

## Parameters
- `commit_message` (required): The commit message

## Examples
```
# Commit with a simple message
gcm "Fix bug in login"

# Commit with a detailed message
gcm "Add new feature: user authentication with OAuth2"
```
