# mkcd - Create directory and change into it

## Description
Creates a new directory (including parent directories if needed) and immediately changes into it.

## Usage
```
mkcd <directory>
```

## Parameters
- `directory` (required): Directory path to create

## Examples
```
# Create and enter a single directory
mkcd myproject

# Create nested directories and enter the deepest one
mkcd projects/web/frontend
```
