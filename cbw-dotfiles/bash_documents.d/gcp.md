# gcp - Git commit and push

## Description
Adds all changes, commits them with a message, and pushes to remote in one command.

## Usage
```
gcp <commit_message>
```

## Parameters
- `commit_message` (required): The commit message

## Examples
```
# Quick commit and push
gcp "Fix login bug"

# Commit and push with detailed message
gcp "Add user authentication feature with OAuth2 support"
```
