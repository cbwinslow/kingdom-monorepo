# glog - Git log in pretty format

## Description
Shows git commit history in a compact, colorful, graph format with branch decorations.

## Usage
```
glog [count]
```

## Parameters
- `count` (optional): Number of commits to show (default: 10)

## Examples
```
# Show last 10 commits
glog

# Show last 20 commits
glog 20

# Show last 5 commits
glog 5
```
